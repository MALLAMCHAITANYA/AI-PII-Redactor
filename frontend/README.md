# AI PII Redactor - Frontend

This is the user interface for the AI PII Redactor, built with **React**, **Vite**, and **Tailwind CSS**. It provides a modern, responsive dashboard for uploading files and viewing redaction results.

## 🛠️ Technologies Used
- **React**: Modern component-based UI library.
- **Vite**: Ultra-fast build tool and development server.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Axios**: Promised-based HTTP client for API requests.

## 🚀 Setup & Execution

### 1. Prerequisites
- **Node.js**: Ensure you have Node.js installed on your system.

### 2. Installation
Navigate to the frontend folder and install the dependencies:
```bash
npm install
```

### 3. Running Development Server
```bash
npm run dev
```
The frontend will start at `http://localhost:5173`.

## 📦 Build for Production
To generate a production-ready build:
```bash
npm run build
```
The output will be in the `dist/` directory.
